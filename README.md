# AIQuiz – Multiplayer Party Game

**AIQuiz** ist ein Multiplayer-Party- & Quizspiel für Couch-Gaming, Smartphone und Desktop.

In verschiedenen Spielmodi – z. B. *Guess the Price*, *Schätzfragen*, *Reaktionsspiele* und vielen weiteren Minigames – treten Spieler gegeneinander an oder im Team.

## Nutzung von Amazon-Produktdaten

Im Spielmodus **„Guess the Price“** werden unter anderem:

- Produktnamen  
- Preise  
- Bewertungen  
- Produktbilder  
- und Links zu Amazon-Produkten  

angezeigt.  

Diese Daten werden über die **Amazon Product Advertising API (PA-API)** bezogen.
Die Produktbilder und Links werden unverändert dargestellt, und beim Klick auf den Link
wird der Nutzer direkt zur entsprechenden Produktseite auf Amazon weitergeleitet.
So erhält der Nutzer eine Breite Auswahl an verschiedenen Produkten. Diese Produkte werden dem Nutzer auf spielerische Art und Weise nähergebracht. 
So erhält er potenziell neue Ideen für Geschenke oder auch für sich selbst.

## Zweck dieser Seite

Diese Seite dient als Informationsseite für das Amazon-Partnerprogramm (Amazon PartnerNet).
Die Amazon-Affiliate-Links werden **innerhalb der App** angezeigt, nicht auf dieser Seite.
